Section Bug 예시

  • section 안/밖에서 사용되는 범위가 잘못된 경우 발생합니다.
  • section 사용범위는 [이곳]에서 확인하세요.

headingArea

  • headingArea는 페이지에서 한번만 쓰이며, section 밖에서 사용됩니다.
  • mTitle > h2는 section당 한번만 사용 되며, h2를 기준으로 section화 하면 됩니다.
  • section의 시작은 항상 mTitle입니다. 다른 태그가 올 경우에도 동일한 버그가 발생합니다.
