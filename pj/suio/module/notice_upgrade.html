upgrade

쇼핑몰 화면에 표시되는 적립금을 편의에 따라
표시할 수 있도록 설정 기능이 업그레이드 되었습니다.

  • 안내기간 : 2013-02-06 (수) ~ 2013-02-19 (화)
  • 변경일 : 2013-02-20 (수)
  • 변경내용 : 주문조회/취소/교환 → 전체주문 조회
  • 업데이트 내용

    적립금 표시방식에서 적립금 단위가 고정이었으나, 적립금 금액의 좌/우로 자유롭게 표시할 수 있게 되었습니다. 또한, 쇼핑몰 전체 화면에 적립금과 단위가 일관되게 표시되도록 개선되었습니다.
