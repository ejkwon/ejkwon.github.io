레시피 채널관리

서비스상태
서비스 상태 도움말
  • 레시피 서비스 상에 채널이 등록되었는지, 사용자가 이용할 수 있는지 구분합니다.
  • '서비스중'인 채널이라도 검색 공개여부 설정에서 이용 권한을 부여한 사용자만
    해당 채널을 이용할 수 있는 점 유의하시기 바랍니다.
서비스 상태 도움말
상태값 설명 서비스 버전
서비스중 레시피 서비스에 채널 등록됨, 사용자에게 제공됨 배포 시 수정됨
준비중 채널 등록 전, 사용자에게 제공되지 않음 없음
배포상태
배포 상태 도움말
  • 레시피 서비스 상에 채널을 제공하기 위해 또는 제공중인 채널에 변경사항이 있을 경우에는 배포 요청을 해야하며, 처리 결과는 상태 값으로 알 수 있습니다.
  • 채널 수정 화면에서 [배포] 버튼을 클릭하면 카페24에 해당 채널의 배포 승인 요청이 접수되고, '심사대기' 상태로 표시됩니다.
  • 배포요청 후 심사중인 단계까지는 채널 정보를 수정할 수 없습니다.
  • 카페24 운영자는 '심사중' 상태로 변경하기 직전인, '심사대기' 상태로 요청된 내용까지 반영된 가장 최신 정보를 기준으로 테스트를 진행하고 배포 상태를 변경합니다.
  • 레시피 서비스 서버에 새 버전으로 '배포완료'되기 전까지, 변경한 내용은 레시피 서비스상에 반영되지 않습니다.
  • 배포완료 이후 채널을 수정하지 않으면 가장 최근 배포 버전에 대한 상태값이 표시되고, 수정하면 다시 배포를 요청하기 전까지는 상태값이 표시되지 않습니다.
