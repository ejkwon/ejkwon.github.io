배포 승인 절차

  1. 1. 심사대기
    • [배포] 버튼을 클릭하면 카페24에 해당 채널의 배포 승인 요청이 접수되고, '심사대기' 상태로 표시됩니다.
    • 레시피 서비스 서버에 '배포완료'되기 전까지, 변경한 내용은 레시피 서비스상에 반영되지 않습니다.
    • 카페24 운영자가 심사를 시작하기 전까지는 [배포 요청 취소]하면 채널 정보를 다시 수정할 수 있습니다.
  2. 2. 심사중
    • 카페24 운영자가 개발사에서 접수한 내용을 검토중인 상태입니다.
    • '심사중' 상태로 변경하기 직전인, '심사대기' 상태로 접수된 가장 최신 정보를 기준으로 테스트를 진행합니다
    • 심사중 상태일 때에는 심사를 마치고 배포 승인 상태가 변경되기 전까지 채널 정보를 수정하거나, 배포 요청을 취소할 수 없습니다.
  3. 3. 배포보류
    • 개발사에서 요청한 배포 버전의 정상 동작 여부를 검토하였으나, 다소 불완전한 부분으로 인해 접수한 내용이 반려된 상태입니다.
    • 개발사에서는 거절 사유를 확인하고 채널 정보를 수정해야 합니다. 수정한 후에도 다시 [배포] 버튼을 클릭하기 전까지는 배포보류 상태가 유지됩니다.
    • 수정한 내용을 반영하시려면 [배포] 버튼을 클릭해주세요.
  4. 4. 배포완료
    • 카페24 운영자가 개발사에서 접수한 내용을 검토한 결과, 정상적으로 채널의 역할을 수행하는 것으로 확인하여 레시피 서비스상의 채널 버전을 업그레이드 했습니다.
    • 채널 정보를 다시 수정하더라도 레시피 서비스 사용자의 채널 이용에는 영향이 없습니다. (가장 최근에 배포 승인된 채널 정보만이 레시피 서비스상에 반영됨)
