App별 판매내역 도움말

  • 표시된 금액과 실제 정산되는 금액의 차이가 발생할 수 있습니다. 실제 정산금액에 대한 확인은 PG사를 통해 확인이 가능합니다.
  • 신규고객수 : 기존에 App을 설치했던 고객이 아닌 App을 신규로 구매하여 설치한 고객 수
  • 설치건수 : 쇼핑몰 운영자가 App을 설치한 총 건수 (유료, 무료 설치 건수를 모두 포함합니다.)
  • 결제금액 : App 판매단가 * 설치건수
  • 환불건수 : App이 실제 환불이 된 건수 (삭제와 무관하며 삭제된 건수 중 실제 환불이 발생한 건수를 카운트 합니다.)
  • 환불금액 : App의 판매가와 무관하게 실제 환불이 된 금액
  • 판매금액 : 결제금액 - 환불금액
  • 수수료 : 판매금액 * 30% (소수점버림)
  • 수익금액 : 판매금액 - 수수료
