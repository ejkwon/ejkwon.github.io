결제(PG) 서비스 진행 안내
진행상황
  • 온라인신청완료 : 결제(PG) 서비스 신청에 필요한 온라인 정보를 PG사에 제출한 상태 입니다.
  • 카드사 심사 진행중 : 온라인 신청, 구비서류, 계약서가 PG사에 서면 접수 되어 카드사 심사가 진행 중인 상태 입니다.
  • 카드사 심사 실패 : 카드사 심사 결과 결제(PG) 서비스 신청 승인이 반려 된 상태 입니다. PG사에 문의 해 주십시오.

구비서류 안내

  • 사업자별 구비서류와 계약서를 작성한 후 PG사에 우편으로 접수 되어야 합니다.
  • 구비서류가 접수 되고 난후 카드사별로 심사가 진행 되며 카드사 심사 통과 후 결제(PG) 서비스 등록이 완료 됩니다.
