API Tester

  1. 01
    카페24에서 제공하는 API를 테스트 하기 위해서 Postman 프로그램의 사용을 권장합니다.
    • Postman 프로그램의 개인적 사용은 무상으로 사용이 가능합니다. (월 1,000회 API 호출가능)
    • 다운로드 및 자세한 설명은 Postman 웹사이트를 참고하세요. Postman 웹사이트 바로가기
  2. 02
    Postman 설치 후 API 를 쉽게 할 수 있도록 제작된 API 컬렉션 파일을 다운로드 합니다.
    • 카페24 API 컬렉션 파일에는 Front, Admin에서 사용하는 대부분의 API를 테스트할 수 있도록 구성되어 있습니다. API 컬렉션 다운로드
  3. 03
    Postman 을 실행한 후 API 컬력션 파일을 import 합니다.
    • Postman 프로그램에서 왼쪽 상단 import 버튼을 이용하여 API 컬렉션 파일을 import 합니다.
    • 정상적으로 import 되면 Collection 에서 API 리스트를 확인할 수 있습니다.
  4. 04
    Postman 의 컬렉션에서 카페24 API 컬렉션을 확인할 수 있으며, 테스트가 가능합니다.
    • 왼쪽 Collections 리스트에서 테스트를 원하는 API를 선택하면 테스트 준비가 완료됩니다.
    • 테스트시 아래와 같은 설정은 변경하셔야 합니다.
      1. 1) 인증키는 선택한 API 하단 Authorization 메뉴에서 oAuth2.0 token 으로 설정하여 입력합니다.
      2. 2) {#id} 로 표시된 부분은 각 Entity별 고유 코드입니다 (ec: 상품이면 상품코드, 회원이면 회원ID를 입력합니다.)
  5. 05
    API의 사용법이나 파라미터 데이터를 자세히 알고 싶으신 경우 API 문서를 통해 확인할 수 있습니다.
    • 카페24 API 문서는 API 가이드 페이지에서 쉽게 확인할 수 있습니다. API 가이드 바로가기
    • Postman 의 사용법이나 자세한 안내가 필요하시면 Postman 공식 웹사이트에서 확인이 가능합니다. Postman 웹사이트 바로가기
