로그인 개발사가입 개발사 어드민

미래의 커머스를
당신만의 앱으로 시작하세요.

개발자센터 소개 및 API 안내입니다.

  • 입점안내 개발자센터에 대한 안내와
    입점절차를 안내합니다.
  • 기본사용안내 개발자센터를 통한 App 등록과
    개발자 어드민의 기본사용방법을 안내합니다.
  • App Store 판매안내 개발사에서 제작한 App를 카페24 App Store에
    판매하기 위해 필요한 정보를 안내합니다.
  • API 가이드 스토어에 대한 경험을 극대화 할 수 있는
    카페24 API 가이드를 안내합니다.
  • API 제공목록 카페24에서 매주 새롭게 업데이트 되는
    API 내역을 확인할 수 있습니다.
  • API Support Tool API 사용에 대한 개발자들의 고민을 쉽게 해결하고
    카페24 API 컬렉션을 쉽게 테스트할 수 있는
    개발도구의 사용방법을 안내합니다.

개발사가 초기에 쉽게 개발할 수 있도록
주요한 개발 정보에 대한 가이드를 제공하고 있습니다.

  • App 개발 시작하기 App을 쉽게 개발하기 위해
    기본적으로 필요한 절차를 습득할 수 있습니다.
  • 인증/보안 가이드 Oauth 2.0 기반의 인증에 대한 개발지식을
    쉽고 빠르게 습득할 수 있습니다.
  • 레퍼런스 소스 실제 개발된 소스코드를 확인하고 따라할 수 있는
    레퍼런스 소스코드를 제공하고 있습니다.

우리의 많은 파트너들은 이미 카페24 App스토어를 통해
매월 높은 수익과 성장을 만들고 있습니다.
