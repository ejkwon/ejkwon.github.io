레시피 서비스 소개
레시피(Recipe) 서비스는

업무 자동화에 필요한 레시피들을 제공하는 일종의 매크로 서비스입니다.

레시피란?

특정 "조건"이 발생될 때 원하는 동작을 "실행"하도록 만들어 놓은 지시문입니다.

  • 예시1 : 페이스북에 사진이 업로드되면, 핀터레스트에 사진을 업로드한다.
  • 예시2 : 매일 오후 3시에, '결제완료' 주문내역을 구글 Sheet에 올린다.
서비스 목적

잘 만들어진 서비스와 플랫폼이 아무리 많더라도 본인에게 필요한 정보가 신속하고,
정확하고, 편리하게 제공되지 않으면 업무를 원활히 처리하기에 어려움이 있습니다.

매일 반복되는 행동을 하고 있는 시간이라도 절약할 수 있다면, 남은 시간은 더 생산적이고 효율적인
일에 투자할 수 있지 않을까요?

이에, 카페24에서는 기존에 사용한 서비스 그대로, 단순히 '연결'하는 행위만을 통해 간편하게
자동으로 업무를 처리하는 플랫폼 서비스 "레시피(Recipe)"를 시작하게 되었습니다.
