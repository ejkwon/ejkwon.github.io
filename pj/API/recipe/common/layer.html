2. 그룹추가 (OR)

그룹A : 다리가 4개인 동물 토끼, 거북이, 개 그룹B : 종류가 파충류인 동물 거북이, 뱀 결과 C : 그룹A 또는 그룹B에 해당하는 것은 토끼, 거북이, 개, 뱀 입니다.
참고사항
  • 필수 조건을 만족한 데이터에 필터 설정이 적용됩니다.
  • 그룹 내의 항목간에도 1번과 같이 AND 조건을 추가할 수 있습니다.
