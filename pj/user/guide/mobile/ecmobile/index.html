VIEWPORT

webkit 기반의 모바일 브라우저들이 모바일에 최적화 된 상태로 화면이 표시 되도록 viewport를 설정합니다.
viewport 설정을 위해 head 태그 안에 meta 태그를 추가합니다.

<!-- common --> <meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=1.0, minimum-scale=1.0, user-scalable=yes"> <!-- zoom --> <meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=2.0, minimum-scale=0.5, user-scalable=yes">
