DTD

DTD는 HTML5로 정의하며 항상 문서의 최상단에 삽입합니다. 마크업은 XHTML 1.0 Transitional 문법과 동일하게 작성합니다.

<!DOCTYPE html>

viewport

webkit 기반의 모바일 브라우저들이 모바일에 최적화 된 상태로 화면이 표시 되도록 viewport를 설정합니다.
viewport 설정을 위해 head 태그 안에 meta 태그를 추가합니다.
User Agent에 따른 뷰포트 설정은 아래와 같습니다.

<!-- iPhone --> <meta content="width=device-width, initial-scale=1.0, maximum-scale=1.0, minimum-scale=1.0, user-scalable=no" name="viewport"> <!-- iPad --> <meta content="initial-scale=1.0; maximum-scale=1.0; minimum-scale=1.0; user-scalable=no;" name="viewport"> <!-- Android --> <meta content="width=device-width, initial-scale=1.0, maximum-scale=1.0, minimum-scale=1.0, user-scalable=no, target-densitydpi=medium-dpi" name="viewport">
