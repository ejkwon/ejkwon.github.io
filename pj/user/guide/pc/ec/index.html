CSS

1. 공통 css + 대메뉴별 css : 각 대메뉴의 css 파일을 생성하여, 공통 css와 함께 선언한다.

공통 : <link rel="stylesheet" href="//img.cafe24.com/img/ec/css/common.css" type="text/css"> - 글로벌 쇼핑몰 : <link rel="stylesheet" href="/img.cafe24.com/img/ec/css/global.css" type="text/css"> - 쇼핑몰센터 안내 : <link rel="stylesheet" href="//img.cafe24.com/img/ec/css/info.css" type="text/css"> - 모바일 쇼핑몰 : <link rel="stylesheet" href="//img.cafe24.com/img/ec/css/mobile.css" type="text/css">

2. 공통 id="container" + 대메뉴별 class : 대메뉴별로 고유한 class를 부여함으로써 구분한다.

- 글로벌 쇼핑몰 : <div id="container" class="globalShop"> - 쇼핑몰센터 안내 : <div id="container" class="infoShop"> - 모바일 쇼핑몰 : <div id="container" class="mobileShop">
