전체 예치금 현황 안내

예치금 입출근 관련 모든 이력이 시간 순서대로 조회 됩니다.

처리구분 안내

  1. 1) 충전 : 상점에서 입금료된 충전 이력
  2. 2) 선지급 : 패스트박스 담당자가 상점의 요청으로 충전된 예치금 외에 일정금액을 배송비로 가산한 이력
  3. 3) 환불 : 상점에서 예치금 환불 요청하여, 패스트박스 담당자가 입금처리완료한 이력
  4. 4) 수기등록 : 패스트박스 담당자가 특정 사유로 예치금을 수정한 이력
  5. 5) 주문출고 : 접수된 주문이 출고오나료 처리되어, 해당 배송비가 예치금에서 차감된 이력
