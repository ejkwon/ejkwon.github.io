수출신고 화주정보 안내

수출신고를 위해 필수적으로 입력해야하는 상점정보 입니다.

1.수출신고 화주정보 항목설명 및 생성 방법

  1. 1) 전자상거래서비스ID : KTNET 간이수출신고 Open API를 이용하기 위한 전자상거래 수출신고 회원 아이디 입니다.

    전자상거래 수출신고 사이트(https://www.goglobal.co.kr) > 회원가입 후 생성됩니다.

  2. 2) 통관고유부호 : 수출입 및 반송신고 또는 환급신청 등의 처리 시 신고인의 관리를 위하여 관세청전산시스템에서 부여하는 부호 입니다.

    관세청 유니패스 Home (https://unipass.customs.go.kr) > 사용자등록 후 생성 됩니다.

  3. 3) 수출신고인부호 : 수출입통관업체 식별을 위하여 통관고유부호체계에 따라 부호관리시스템에서 부여한 부호 입니다.

    관세청 유니패스 Home (https://unipass.customs.go.kr) > 업무처리 > 수입통관 > 통관고유부호 메뉴에서 생성 하실 수 있습니다.

  4. 4) 간이수출신고 인증키 : KTNET 간이수출신고 Open API를 이용하기 위한 인증키 입니다.

    전자상거래 수출신고 사이트(https://www.goglobal.co.kr) > Open API 안내 > Open API 인증키 발급 메뉴에서 생성 하실 수 있습니다.

  5. 5) 기본상품중량(kg) : 수출신고 시 개별 상품의 무게정보로 사용됩니다. 배송사 실측무게와 차이가 발생할 수 있습니다.
