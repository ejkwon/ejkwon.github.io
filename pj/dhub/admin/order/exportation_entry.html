수출신고관리 안내

1. 수출신고상태 안내

  1. 1) 미신고 : 수출신고요청 전 주문
  2. 2) 신고요청 : 수출신고요청 후 관세청에서 발급한 수출신고번호를 회신 받기 전 주문
  3. 3) 신고처리중 : 수출신고요청 후 관세청에서 수출신고처리중인 주문
  4. 4) 신고실패 : 수출신고요청 후 수출신고처리 중 오류가 발생한 주문 (KTnet 확인 필요 고객지원센터 02-6000-2169)
  5. 5) 신고완료 : 수출신고요청 후 관세청에서 발급한 수출신고번호를 회신 받은 주문
  6. 6) 이행완료 : 수출신고완료된 주문이 배송사를 통하여 수출이행처리된 주문
  7. 7) 신고취하요청 : 수출신고취하요청 후 관세청에서 발급한 수출신고승인처리 결과를 받기 전 주문
  8. 8) 신고취하실패 : 수출신고취하요청 후 수출신고취하처리 중 오류가 발생한 주문 (KTnet 확인 필요 고객지원센터 02-6000-2169)
  9. 9) 신고취하승인 : 수출신고취하요청 후 관세청에서 발급한 수출신고승인처리 결과를 받은 주문

2. 확인필요 주문현황 안내

  1. 1) 수출신고요청 필요건수 : 수출신고요청이 필요한 주문 건수
  2. 2) 배송사 이행신고 확인필요건수: 수출신고완료되어 이행신고 미완료 상태 주문 건수
  3. 3) 수출취하요청 필요건수 : 출고취소상태의 수출신고요청/신고완료 상태 주문 건수
  4. 4) 관세청 취하승인 필요건수 : 수출신고취하요청되어 수출신고승인 미처리 상태 주문 건수
  5. 5) KTnet 확인 필요건수 : 수출신고요청실패/수출신고취하실패 상태의 주문 건수
