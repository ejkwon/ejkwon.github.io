수출신고관리 안내

  • 수출신고상태 안내
    1. 1) 미신고 : 수출신고 전 주문
    2. 2) 신고완료 : 수출신고요청 후 관세청에서 발급한 수출신고번호를 회신 받은 주문
    3. 3) 신고실패 : 수출신고요청 후 수출신고처리 중 오류가 발생한 주문 (패스트박스에서 확인중인 주문)
    4. 4) 정정완료 : 수출신고완료 후 정정완료 되어 수출신고가 완료된 주문
    5. 5) 취하요청 : 수출신고완료 후 취하 요청해 놓은 주문(수출신고 취하 진행중인 주문)
    6. 6) 취하완료 : 수출신고취하요청 후 취하가 완료된 주문
