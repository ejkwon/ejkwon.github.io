바코드 스캔

바코드 배송지시 1. 바코드 배송지시 방법
  1. ① 아웃박스 택배사를 선택하거나, 직접 입력합니다.
  2. ② 아웃박스 송장/FB송장번호 행에 아웃박스 송장 번호를 스캔합니다.
  3. ③ 입력된 아웃박스에 들어갈 FB송장번호를 스캔합니다.
    (접수확정 상태의 FB송장번호만 배송지시할 수 있습니다.)
  4. ④ 아웃박스를 변경해야할 때, "COMPLETE"를 입력하고 ①번부터 계속 실행합니다.
2. 이미 스캔한 아웃박스에 FB송장번호 추가
  1. ① "COMPLETE"를 입력합니다.
  2. ② 이미 스캔한 아웃박스와 같은 택배사를 선택 혹은 입력하고,
    아웃박스 송장번호를 스캔합니다.
  3. ③ 추가해야할 FB송장번호를 스캔합니다.
  4. ④ 추가해야할 FB송장번호를 모두 스캔하고 나서 COMPLETE를 입력합니다.
3. 이미 스캔한 아웃박스 혹은 FB송장번호 삭제
  1. 1) 방법1 : 바코드 스캔내역에서 [X] 버튼을 클릭한다.
  2. 2) 방법2
    1. ① "DELETE"를 입력합니다.
      작업을 취소하려면 CANCEL을 입력합니다.
    2. ② 삭제해야할 FB송장번호 혹은 아웃박스 송장번호를 입력합니다.
      예) FB송장번호 2개를 삭제해야할때, 스캔(입력)순서
      DELETE --> FB송장번호1 --> DELETE --> FB송장번호2
4. 이미 스캔한 FB송장번호 이동
  1. ① "MOVE"를 입력합니다. 작업을 취소하려면 CANCEL을 입력합니다.
  2. ② 이동할 FB송장번호를 입력합니다. 작업을 취소하려면 CANCEL을 입력합니다.
  3. ③ 이동할 아웃박스 송장번호를 입력합니다.
    예) FB송장번호1,2를 아웃박스 송장번호 2로 이동 해야할 때,
    MOVE --> FB송장번호1 --> 아웃박스 송장번호 2
    --> MOVE --> FB송장번호2 --> 아웃박스 송장번호 2
  • 아웃박스 송장번호를 스캔하고 인박스송장번호를 스캔합니다.
  • 배송지시처리시 접수확정상태인 주문 내역만 처리됩니다.
  • 아웃박스 송장번호 리셋 처리시 COMPLETE 바코드를 스캔하시기 바랍니다.
  • 아웃박스 송장번호 재 스캔시 (추가) 문구가 표시 됩니다.
