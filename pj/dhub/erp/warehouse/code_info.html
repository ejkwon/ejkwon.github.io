위치코드 정보 안내

  • 위치코드는 존-로케이션-단을 대시(-)로 구분하여 이루어 집니다.
  • 존-로케이션-단의 3개 항목은 각각 1~3자리로 입력해 주세요.
    (예: A-1-1 / A-123-AY / AAA-123-KP3)
  • 위치코드는 영문대문자, 숫자, 대시(-)만 입력 가능합니다.
  • 위치코드 개별등록은 한 번에 20개까지 저장이 가능하며, 대량으로 한 번에 등록하려면 [엑셀등록 바로가기]을 이용해 주세요.
  • 체크박스가 선택된 위치코드만 저장됩니다.
