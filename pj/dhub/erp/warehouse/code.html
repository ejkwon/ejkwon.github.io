위치코드 안내

  • 위치코드는 대시(-)구분자로 존, 로케이션, 단이 조합되어 이루어 집니다.
    (예: A-1-1 / A-123-AY / AAA-123-KP3)
  • 위치코드는 영문대문자, 숫자, 대시(-)만 입력 가능합니다.
  • 위치코드 개별등록은 한 번에 20개까지 저장이 가능하며, 대량으로 한 번에 등록하려면 [위치코드 엑셀업로드 바로가기]를 이용해 주세요.
  • 3PL상품을 이용하는 상점에 설정된 '위치코드 존' 하위 모든 위치코드는 3PL전용이며, 상품별로 위치코드를 선택합니다.
  • 배송대행 존에 설정된 '위치코드 존' 하위 모든 위치코드는 배송대행전용이며, 주문서별로 위치코드를 선택합니다.
  • 3PL전용 위치코드의 이용여부는 다음 기준에 의하여 표시됩니다.
    • 이용중 : 상품별 위치코드 선택됨
    • 비어있음 : 상품에 선택되지 않아 이용가능한 위치코드
    • 삭제됨 : 삭제하여 이용할 수 없는 위치코드
  • 배송대행전용 위치코드의 이용여부는 다음 기준에 의하여 표시됩니다.
    • 이용중 : 배송대행지시서에서 주문서별 위치코드 선택 후 ~ 주문서 출고완료
    • 비어있음 : 주문서에 선택되지 않아 이용가능한 위치코드
    • 삭제됨 : 삭제하여 이용할 수 없는 위치코드
