바코드 스캔

바코드 인계증 생성 인계증 생성 방법
  1. 1. 화물인계증 생성 화면 로드 시 인계증번호를 생성 합니다.
  2. 2. 인박스 송장이 투입 될 아웃박스를 스캔 합니다.
  3. 3. 인박스 송장을 스캔합니다.
  4. 4. 아웃박스를 변경해야할 때, "COMPLETE"를 입력하고 새로운 아웃박스 번호를 스캔 합니다.
    또는 다음작업 대상 아웃박스를 연속 2회 스캔 합니다.
  5. 5. 새로운 아웃박스번호가 생성되면 3번 부터 반복 합니다.
  6. 6. 작업이 완료되면 스캔완료 버튼을 선택 하여 화물인계증을 생성 합니다.
인박스 송장번호 삭제
  1. 1. "DELETE"를 입력합니다.
  2. 2. 삭제해야할 인박스 송장번호를 스캔 합니다.
  3. 3. 스캔한 인박스 송장은 삭제 됩니다.
  4. 4. 추가로 삭제가 필요하면 ①번부터 반복 합니다.
아웃박스 번호 삭제
  1. 1. "DELETE"를 입력합니다.
  2. 2. 삭제해야할 아웃박스 번호를 스캔 합니다.
  3. 3. 아웃박스 번호와 해당 아웃박스에 포함된 인박스 송장이 모두 삭제 됩니다.
  4. 4. 추가로 삭제가 필요하면 1번부터 반복 합니다.
이미 스캔한 인박스송장을 다른 아웃박스로 이동
  1. 1. "MOVE"를 입력합니다.
  2. 2. 인박스 송장이 이동 해야 될 아웃박스를 스캔 합니다.
  3. 3. 이동할 인박스 송장을 스캔하면, 인박스 송장이 이동 합니다.
  4. 4. 추가로 이동이 필요하면 1번부터 반복 합니다.
스캔된 인박스 송장의 배송사 불일치
  1. 1. 인박스 송장 스캔 시 배송사를 확인합니다.
  2. 2. 배송사 불일치 건을 바코드 스캔 결과에 출력 됩니다.
  3. 3. 배송사 불일치 인박스 송장은 등록이 불가능 합니다.
스캔된 인박스 송장의 상점 불일치
  1. 1. 인박스 송장 스캔 시 상점을 확인 합니다.
  2. 2. 배송사 NTI의 경우는 수출신고여부 Y인 경우 동일 상점만 등록 됩니다.
  3. 3. 배송사 NTI의 경우는 수출신고여부 N인 경우 다른 상점도 등록 됩니다.
