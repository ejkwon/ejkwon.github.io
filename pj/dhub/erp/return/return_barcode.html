바코드 반송지시 1. 바코드 반송지시 방법
  1. ① 바코드 반송지시 화면 로드 시 반송지시번호를 생성 합니다.
  2. ② FB송장을 스캔하면서 아웃박스에 투입합니다.
  3. ③ 아웃박스를 변경해야할 때, "COMPLETE"를 입력하면 새로운 반송지시 번호가 생성 됩니다.
  4. ④ 새로운 반송지시번호가 생성되면 ② 번 부터 반복 합니다.
  5. ⑤ 작업이 완료되면 완료 버튼을 선택 하여 바코드 반송지시를 완료 합니다.
2. 인박스 송장번호 삭제
  1. ① "DELETE"를 입력합니다.
  2. ② 삭제해야할 FB송장번호를 스캔 합니다.
  3. ③ 스캔한 FB송장은 삭제 됩니다.
  4. ④ 추가로 삭제가 필요하면 ① 번부터 반복 합니다.
