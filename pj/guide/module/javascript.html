[mTab]

  • <script type="text/javascript"> var call_eTab = function() { $.eTab($(".탭클래스명")); } </script>
  • 위와 같이 .탭클래스를 모두 선택하여 eTab함수의 인수로 넘겨 준다.
  • 한페이지에 탭이 1~n개가 들어가도 위의 함수 호출은 1회만 한다.
  • 숨김/보임 처리되는 컨텐츠는 공통의 class를 갖는다.
  • 보여질 첫번재 컨텐츠는 display:block속성을 지정하고, 형제는 display:none을 지정한다.
  • 컨텐츠내용 중 ol의 list-style을 입힐 경우 IE버그가 있으므로 주의한다. : 참고사이트 가기
