mToggle

  • 컨텐츠 요소가 길어졌을 때, mBorad의 내용을 toggle
  • 열기/닫기에 대한 택스트를 가변으로 적용 최대 한글 11자까지 가능
  • 테이블 및 리스트들과의 보더겹침문제는 인접선택자를 사용하여 .mToggle + .toggleArea .mBoard { margin:-1px 0 0 0; }의 형식으로 처리
  • mTitle에 h2를 제외한 p, ul 등의 요소가 들어갈 경우 디자인이 틀어져 보일 수 있음
  • 다른 Acc 요소들 사용은 고려되지 않음
  • 토글영역을 잡아주기 위해, toogleArea로 그룹핑
