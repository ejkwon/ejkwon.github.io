매뉴얼

[mPicker]

  • 예외케이스로 .section 으로 그룹핑 하지 않는다.
  • 팝업에 사용될 경우 하단 mCtrl삭제를 제외한 마크업 동일
  • 페이징, 간편순서 설정 등의 요소 미사용으로 height값 조정이 필요할 경우 pickerArea에 클래스를 추가하여 조정
    ex) <div class="pickerArea g[class]">
  • height 조정이 필요한 요소 예시
    1. .pickerArea { height:값; }
    2. .mPicker .gTableMerge .mBoard.typeBody { height:값; }
    3. .mPicker.gReverse .gTableMerge .mBoard.typeBody { height:값; }
