[mDisplay] : 2012-12-10 이후 .mPicker로 대체

  • .mDisplay, .gDisplay 사용. 예외케이스로 .section 으로 그룹핑 하지 않는다.
  • .mDisplay 이후에는 다른 페이지와 동일하게 .section 으로 마크업한다.
  • .mPaging은 경우에 따라 추가,삭제 가능
  • 팝업에 사용될 경우 : 하단 gCtrl삭제를 제외한 마크업 동일
  • 페이징, 간편순서 설정 등의 요소 미사용으로 인해 좌,우측 height값이 안 맞을 경우 gDisplay에 클래스 추가하여 typeBody의 height값을 조정한다. ex) <div class=""gDisplay typeSearch typeMail">
  • mDisplay의 hieght값 변경으로 중간 버튼영역(.prdAdd)의 높이값 조정 필요시 .mDisplay + gHeight숫자로 제어 ex) <div class="mDisplay gHeight1">
  • typeBody, 버튼영역 높이값을 모두 변경 시 mDisplay에 클래스 추가하여 조정 ex) <div class="mDisplay gCouponIssue">
