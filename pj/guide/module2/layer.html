[mNotice] 공지사항

1.9에 레이어팝업만 들어갈 경우 샘플
  • 추가되는 이미지(타이틀) 경로 : /img/notice/
  • 이미지 네이밍에 서비스별 prefix 삽입
  • ec관련 타이틀 : eYYYYMMDD (예 : e20121031)
  • ec관련 타이틀이 날자별로 2개 이상일때 : eYYYYMMDD_1 (예 : e20121031_1)
  • globalc관련 타이틀 : gYYYYMMDD (예 : g20121031)
  • 공통 타이틀 : c네이밍 (예 : cName)
