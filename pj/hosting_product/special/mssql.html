제공 환경

  • MS-SQL DB 호스팅 소개
    • Windows Server Core 지원을 통해 OS 패치와 관련된 돌발적인 중단 시간을 줄일 수 있으며
      OS 재부팅이 50-60% 감소.MS-SQL 2008 R2 64bit 버전만 지원합니다.
    • SSD 디스크만으로 서비스합니다.
    • MS-SQL 정품 버전을 사용합니다.
  • 서버환경
    • MS-SQL 2008 R2 64bit
  • 저장공간
    • SSD 사용으로 최고의 퍼포먼스를 제공합니다.
    • DB 공간만 제공하며, 그 외 호스팅용 공간은 제공하지 않습니다.
  • Max User Connection (동시 접속 허용 수)
    • 제한이 없습니다.(Unlimited)

  • 과다한 테이블 개수나 쿼리로 서버에 부하를 발생할 경우, 서비스 이용에 제한이 있을 수 있습니다.
추천 고객
  • MS-SQL 라이선스 비용이 부담스러우신 분
  • DB관리에 자신이 없으신 분
  • 웹 서버와 별도로 DB를 구축하여 사용하시는 분
  • 대용량 순수 DB공간이 필요하신 분
  • 하나의 DB로 여러 개의 홈페이지를 운영하시는 분
