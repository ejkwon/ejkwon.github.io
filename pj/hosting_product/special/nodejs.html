제공 환경

  • node. JS 소개
    • node.js는 Javascript의 일종이지만, 기존의 Javascript와는 개념이 조금 다른 언어입니다.
    • node.js는 서버사이드에서 동작이 가능한 언어로, Javascript의 표준 라이브러리 프로젝트인 CommonJS의 모듈 시스템을 지원하고 있으며, 구글 V8 Javascript를 지원하는 서버사이드 개발 언어/웹서버 입니다.
    • 또한, node.js는 이벤트를 기반으로 비동기 동작하는 non-blocking single thread 언어로, Apache와 같이 모든 요청마다 시스템 쓰레드를 생성하는 시스템에 비해 메모리 사용량이 현저히 적고 강력한 성능을 발휘합니다.
  • 서버환경
    • v6.11.3 / v8.5.0 택일
    • MySQL 5.x UTF-8 전용 (euc-kr은 지원하지 않습니다.)
  • 앱 생성 및 무료도메인 제공
    • 앱 생성은 나의서비스관리 메뉴에서 가능하며, 접속을 위해 appname.cafe24app.com 형태의 무료 도메인을 제공합니다.
  • 앱 도메인 연결
    • 앱당 1개의 보유도메인을 연결할 수 있습니다.

제공 기능

  • node.js 호스팅 장점
  • 이메일 계정
  • MySQL 웹어드민 제공
  • 고객님이 원하는 시간에 직접 시작/중지가 가능하여 수정된 사항이 사이트에 즉시 반영됩니다.
  • node.js stable 버전인 0.6.13 버전을 제공하여 최신의 기술을 마음껏 이용할 수 있습니다.
  • 제공된 메모리 용량 내에서 앱을 개발/구동하며 독립적인 컨테이너로 작동하여 서비스 구동 안정성이 보장됩니다.
  • git를 이용한 소스 관리가 가능하여 클라이언트에서 손쉽게 수정 후 업데이트 할 수 있습니다.
  • POP3 메일계정은 지원하지 않으며 서비스 세팅 후 나의서비스관리에서 무료 웹메일을 신청할 수 있습니다.
  • 고객님들이 간편하고 쉽게 DB를 관리하실 수 있도록 MySQL 웹어드민을 제공합니다.
  • MySQL이란 리눅스 기반의 호스팅 서버에서 사용되는 DB(데이터베이스) 입니다.
