제공 환경

  • Tomcat JSP 광호스팅소개
    • Java 기반의 웹사이트 개발기술인 JSP 이용을 위한 웹호스팅입니다.
    • JSP(Java Server Pages)는 Java 기반의 기술로 특정 플랫폼에 제한되지 않고 Java가 가지고 있는 효율성, 메모리 관리 등의 장점을 그대로 가지고 있는 강력한 언어입니다.
    • Tomcat 8.0 최신 버전 지원 및 단독 톰캣 서버 제공으로 보다 안정된 JSP호스팅을 제공합니다.
    • 절약형부터 자이언트형까지 모든 상품에 단독톰캣서버를 제공합니다.
  • 서버환경
    • 기본 : UTF-8, MySQL5.5.X 지원
    • 개발언어 : JSP
    • Tomcat version
      • Tomcat 8.0.x (jsp 2.3/Servlet 3.1)
      • Tomcat 7.0.x (jsp 2.2/Servlet 3.0)
    • JDK : 1.8.x/1.7.x
  • 톰캣서버 안내
    • 톰캣서버의 설정 파일 내(server.xml) 셋팅된 포트번호는 임의적인 수정을 지원하지 않습니다.
    • 임의로 수정한 경우에는 톰캣서버를 정상적으로 이용하실 수 없습니다.
  • 무료 도메인 제공
    • 호스팅에 연결되는 아이디를 이용한 무료 도메인을 제공합니다.
    • 호스팅 신청 아이디가 myname인 경우 myname.cafe24.com 도메인을 무료로 제공해 드립니다.
  • SFTP/FTP/SSH 지원 도움말
    • SSH 터널링(Tunneling)은 지원하지 않습니다.
  • Language/Script
    • Jsp, Java(Servlet), Perl, gcc, C/C++ complier, Python
    • 단, PHP는 지원하지 않습니다.

  • 단독 톰캣
    서버 장점
  • 타임머신
    백업/복구
  • 이메일 계정
  • 웹FTP 제공
  • MySQL
    웹어드민 제공
  • 고객님이 원하는 시간에 직접 시작/중지가 가능하여 수정된 사항이 사이트에 즉시 반영됩니다.
  • 톰캣 프로세스를 단독으로 사용할 수 있어 서비스가 더욱 안정적입니다.
  • 서블릿 재구동을 직접할 수 있습니다.
  • 공용서버의 주기적인 톰캣 재시작으로 인한 불편함이 없습니다.
  • 수정된 클래스 파일을 적용하기 위해 톰캣이 재시작되는 특정 시간을 기다려야 하는 공용 서버의 불편함이 없습니다.
  • 관리 편의를 위해 Tomcat Web Application Manager와 Virtual Host Manager를 지원합니다.
  • DATA와 DB의 복원/백업 서비스는 금일 새벽 05시에 백업된 자료를 포함하여 1일 전 ~ 7일 전 중 원하는 날짜로
    복원/백업이 가능한 서비스입니다.
  • [나의서비스관리 > 계정관리 > DATA&DB 복원/백업] 메뉴를 통해 이용할 수 있습니다.
  • 각 계정의 메일 용량은 5M이며, 메일 발송은 한번에 30통/ 하루에 500통까지만 발송이 가능합니다.
  • 500통이 초과되면 발송은 차단되고, 자정(00시)에 초기화 됩니다.
  • 일반 FTP 프로그램에 익숙하지 않은 고객님들을 위해 간편하고 사용하기 쉬운 웹FTP를 지원합니다.
  • FTP란 호스팅을 받는 서버에 접근하여 파일을 업로드하거나 다운로드 할 수 있는 툴(TOOL)입니다.

월사용료 혜택
  • 신청하시는 서비스 기간에 따라 월사용료 할인 혜택을 제공합니다.
    6개월(5%), 12개월(10%), 24개월(12%), 36개월(15%)
서비스 변경/추가
  • 서비스를 사용하다 용량이 부족한 경우, 상위 서비스로의 변경이 가능합니다.
    (하위 서비스로의 변경은 불가능)
  • 하드와 트래픽 용량의 스트리밍/CDN은 나의서비스관리에서 별도 신청할 수 있으며 CDN 서비스는 이미지만 가능합니다.
서비스 제한
  • 서버 부하가 과도한 계정은 서비스가 취소될 수 있습니다.
  • MySQL DB용량은 서버 디스크 한도 내에서 무제한으로 제공되며 과도한 이용 시 제한될 수 있습니다.
  • 무료 제공되는 스트리밍 서비스는 트래픽 폭주 시 일시적으로 서비스를 제한할 수 있습니다.

  • 웹메일 무료제공

    별도의 사이트 개설이나 메일 서버 구축 없이 고객님의 도메인으로 고유 메일 주소를 이용할 수 있습니다. 고객님의 고유 메일 주소는 대형 포털에서 발급해 주는 무료 메일 주소가 할 수 없는 기업의 홍보역할을 합니다.

  • 그룹웨어 무료제공

    웹메일, 전자결재, 전자게시판, 업무관리 등 통합적 업무관리를 할 수 있는 Smart Work 솔루션을 경험할 수 있습니다. 추가 비용 없이, 우리 회사만의 메신저(Office Talk)를 함께 사용할 수 있습니다.

  • 이미지 CDN 무료 제공 도움말

    카페24 호스팅을 신청하시면 무료로 이미지 CDN 서비스가 제공되어 사실적 용량 증가 효과가 있습니다. 트래픽이 많이 발생하는 이미지를 무료 이미지 CDN 서버에서 제공함으로써 웹 페이지에 집중되는 트래픽을 분산시킬 수 있습니다.

  • 스트리밍 무료 제공 도움말

    카페24 호스팅을 신청하시면 무료로 스트리밍 서비스가 제공되어 사실적 용량 증가 효과가 있습니다. 음악파일 또는 동영상으로 인한 트래픽 증가로 스트리밍 서비스가 차단되더라도 사이트의 차단없이 안정적으로 운영하실 수 있습니다.

  • 웹로그 제공

    방문자 분석에서부터 광고효과분석까지 가능한 웹로그분석 ASP 서비스를 무료로 제공해 드립니다. 카페24 웹로그를 통해 지속적인 고객분석을 통해 효율적으로 사이트를 운영하세요.
