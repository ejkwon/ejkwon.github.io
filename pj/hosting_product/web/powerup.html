제공 환경

  • 서버환경
    • Apache 2.0
    • PHP 7.0
    • MariaDB-10.0.x
    • HTTP/2
  • 무료 도메인 제공
    • 호스팅에 연결되는 아이디를 이용한 무료 도메인을 제공합니다.
    • 호스팅 신청 아이디가 myname인 경우 myname.cafe24.com 도메인을 무료로 제공해 드립니다.
  • MariaDB 무제한 제공
    • MySQL보다 빠른 성능, 높은 호환성의 관계형 데이터 베이스
    • MariaDB는 오픈 소스의 관계형 데이터베이스 관리 시스템(RDBMS)으로 MySQL 대비 최고 70% 이상의 성능이 향상되었고, 새로운 저장 엔진인 아리아(Aria)뿐만 아니라 InnoDB를 교체할 수 있는 XtraDB 저장 엔진을 포함합니다.
    • MySQL과 동일한 소스 코드를 기반으로 하여, MySQL이 가지고 있는 모든 기능을 완벽히 구현하여 MySQL과 높은 호환성을 유지합니다.
  • SFTP/FTP 지원 도움말
    • SSH/Telnet은 지원하지 않습니다.
  • SSL 향상
    • SNI 지원합니다.
    • SSL 기본포트 (443) 사용가능합니다.
  • Language/Script
    • cgi, perl script, SSI, Flash, DBI/DBD, zend optimizer, Python, GD라이브러리 등
      이외의 다른 데몬, lib나 파일을 설치해서 사용할 시에는 작동을 안하거나 비정상적으로 작동할 수 있습니다.
  • 트래픽 안내
    • 트래픽 차단 걱정 없이 안정적인 사이트 운영
    • 사이트 접속자 증가에 따른 사이트 접속 불가, 접속 지연 없이 운영할 수 있습니다.

자세히보기
  • DATA와 DB의 복원/백업 서비스는 금일 새벽 05시에 백업된 자료를 포함하여 1일 전 ~ 7일 전 중 원하는 날짜로
    복원/백업이 가능한 서비스입니다.
  • [나의서비스관리 > 계정관리 > DATA&DB 복원/백업] 메뉴를 통해 이용할 수 있습니다.
  • MariaDB는 Localhost로 기본 접속되며 [나의서비스관리 > MariaDB 외부 IP 접근설정] 을 통해 등록한 IP로 외부접근이 가능합니다.
  • 파워업 무제한 광호스팅은 로컬 POP3 메일 계정이 제공되지 않습니다.
  • 이메일 사용은 무료 웹메일 또는 무료 그룹웨어 신청 후 이용하실 수 있습니다.
  • 일반 FTP 프로그램에 익숙하지 않은 고객님들을 위해 간편하고 사용하기 쉬운 웹FTP를 지원합니다.
  • FTP란 호스팅을 받는 서버에 접근하여 파일을 업로드하거나 다운로드 할 수 있는 툴(TOOL)입니다.
자세히보기
  • 고객님들이 간편하고 쉽게 DB를 관리하실 수 있도록 MariaDB 웹어드민을 제공합니다.
  • MariaDB는 MySQL과 호환되는 리눅스 기반의 호스팅 서버에서 사용되는 DB(데이터베이스) 입니다.
  • 리눅스 서버에는 웹방화벽(ModSecurity)을 윈도우 서버에는웹나이트(WebKnig-ht)를 설치해 드립니다.
    • 웹방화벽(ModSecurity)
      • Apache 웹 서버 기반의 오픈 소스 방화벽으로 PHP Injection및 SQL Injection, XSS 등의 다양한 웹 공격 및 악의적인 접속요청이 발견되면 차단, 로깅 등의 사전에 정의된 행위를 수행하므로 안전한 호스팅 서비스를 받으실 수 있습니다.
    • 웹나이트(WebKinght)
      • IIS 웹 서버에 설치되는 공개용 웹방화벽으로 ISAPI 필터 형태로 동작하며, IIS 서버 최앞단에 위치하여 패킷이
        웹서버로 접근하기 전에 필터 룰에 따라 검증을 하고 SQL Injection공격 등 특정 웹 요청을 사전에 차단함으로서
        웹서버의 보안을 한층 더 강화할 수 있습니다.
  • 고객님의 사이트를 안전하게 지켜드리는 웹방화벽으로 안전한 호스팅 서비스를 받으실 수 있습니다.
자세히보기
  • 호스팅을 신청하시면 인공지능 스팸필터가 지긋지긋한 스팸 게시물에서 해방시켜 드립니다.
  • 제로보드, 테터툴즈 뿐만 아니라 그외 어느 게시판을 클릭 한번으로 스팸필터를 작동시킬 수 있어 편리합니다.

  • 하드용량은 [나의서비스관리 > 추가옵션신청] 메뉴에서 별도 신청할 수 있습니다.
서비스 제한
  • 트래픽 무제한 서비스는 홈페이지 운영목적으로만 이용이 가능합니다.
  • 아래 사항에 해당될 경우, 서비스 이용에 제한이 있을 수 있습니다.
    • 웹사이트 운영 전용이며 보편적인 홈페이지 운영 이외의 용도로 사용하는 경우 (게임/사행성 사이트등)
    • 사이트의 기능 또는 프로그램에 의해 서버 부하를 발생하는 경우
      (이미지/동영상의 업/다운로드/스트리밍,방송,채팅,게임,파일 백업/공유/배포,팬클럽 운영 등)
    • 일반적인 웹사이트에 비해 서버 자원 과다 점유 또는 부하발생, DDoS 등으로 인하여 동일서버 내의 다른 고객에게 피해를 주는 경우
    • 오픈마켓의 이미지/음악/동영상 등의 링크는 제한되며, 미디어 파일(음악/동영상) 또한 외부 링크는 이용하실 수 없습니다.
추천 고객
  • 트래픽 걱정 없이 안정적인 웹사이트 운영을 원하시는 분
  • 정기적인 이벤트로 이벤트 트래픽이 증가 하는 분
  • 갑작스런 트래픽 증가로 사이트 차단이 걱정 되시는분
  • 가끔 있는 트래픽 초과때문에 트래픽 증설하기는 아까우신 분

  • 웹메일 무료제공

    별도의 사이트 개설이나 메일 서버 구축 없이 고객님의 도메인으로 고유 메일 주소를 이용할 수 있습니다. 고객님의 고유 메일 주소는 대형 포털에서 발급해 주는 무료 메일 주소가 할 수 없는 기업의 홍보역할을 합니다.

  • 그룹웨어 무료제공

    웹메일, 전자결재, 전자게시판, 업무관리 등 통합적 업무관리를 할 수 있는 Smart Work 솔루션을 경험할 수 있습니다. 추가 비용 없이, 우리 회사만의 메신저(Office Talk)를 함께 사용할 수 있습니다.

  • 웹로그 제공

    방문자 분석에서부터 광고효과분석까지 가능한 웹로그분석 ASP 서비스를 무료로 제공해 드립니다. 카페24 웹로그를 통해 지속적인 고객분석을 통해 효율적으로 사이트를 운영하세요.
