제공 환경

  • 서버환경
    • 운영체계 : Windows Server 2016 64bit
    • DB : MS-SQL Server 2016 64bit
    • NET Framework : 4.0
  • 무료 도메인 제공
    • 호스팅에 연결되는 아이디를 이용한 무료 도메인을 제공합니다.
    • 호스팅 신청 아이디가 myname인 경우 myname.cafe24.com 도메인을 무료로 제공해 드립니다.
  • Language/Script
    • ASP, .NET, C#, XML, HTML, Flash, Javascript, Vbscript (PHP, CGI 사용불가)
  • FTP 지원 도움말
    • 계정 아이디별로 포트를 별도 할당합니다. Remote Desktop은 지원하지 않습니다.
  • MS-SQL 접속
    • SQL Server Management Studio로 접속하셔야 합니다.
  • 웹(http)으로 접근
    • 반드시 www 폴더 내에 업로드 또는 새폴더를 생성해 주셔야 합니다.
  • 컴포넌트
    • Upload 컴포넌트 : DEXTUpload Pro x64
    • 실명인증 컴포넌트 : 한국신용평가
  • 결제모듈 지원
    • INICIS 5.0 / KCP / LG유플러스 지원하며, 기타 모듈의 지원 여부는 문의 바랍니다.

제공 기능

  • HTTP/2 지원
  • AlwaysOn
  • 열 단위 쿼리
    가속기
  • SQL Server Tools
  • IT 관리 개선
  • 이메일
    계정
  • 웹FTP 제공
  • 웹방화벽
    제공
  • HTTP/2 는 구글의 SPDY라는 통신 프로토콜을 기반으로 개발한 표준이며, 웹페이지 로딩 시간을 단축하기 위해 사용하는 기술입니다.
  • HTTP/2는 HTTP/1.1과 호환성을 유지하면서, 브라우저 - 서버 간 연결에 여러 가상 통신회로를 마련하여 양방향 통신을 실시, 많은 요청을 동시에 전송할 수 있습니다.
  • Windows Server Core 지원을 통해 OS 패치와 관련된 돌발적인 중단 시간을 줄일 수 있으며 OS 재부팅이 50-60% 감소합니다.
  • 쿼리 성능을 10배에 이르기까지 크게 높일 수 있고, 빠른 응답 시간으로 데이터를 대화식으로 사용하여 성능 튜닝을 줄였습니다.
  • 통합된 개발자 환경(SQL Server Tools)로서 선언적 데이터베이스 정의, 기존 데이터베이스 동적 편집,
    라운드트립 코드 리팩터링 및 대상화된 데이터베이스 구축이 가능합니다.
  • FileStream 및 전체 텍스트 검색, 2D 공간 데이터 지원 및 FileTable 성능이 향상된 더욱 유연하고 혁신적인
    응용 프로그램 개발 가능합니다.
  • 응용 프로그램 테스트를 단순화하고 분산 재생을 통해 응용 프로그램 변경 및 업그레이드에 따른 오류 최소화가 가능합니다.
  • 각 계정의 메일 용량은 20M이며, 메일 발송은 한번에 30통/ 하루에 500통까지만 발송이 가능합니다.
  • 500통이 초과되면 발송은 차단되고, 자정(00시)에 초기화 됩니다.
  • 일반 FTP 프로그램에 익숙하지 않은 고객님들을 위해 간편하고 사용하기 쉬운 웹FTP를 지원합니다.
  • FTP란 호스팅을 받는 서버에 접근하여 파일을 업로드하거나 다운로드 할 수 있는 툴(TOOL)입니다.
자세히보기
  • 윈도우 광호스팅(SQL2016)은 웹방화벽(ModSecurity)을 설치해 드립니다
  • 오픈 소스 기반의 방화벽인 ModSecurity는 PHP Injection및 SQL Injection, XSS 등의 다양한 웹 공격 및 악의적인 접속요청이 발견되면 차단, 로깅 등의 사전에 정의된 행위를 수행하므로 안전한 호스팅 서비스를 받으실 수 있습니다.

  • 서버 부하가 과도한 계정의 경우 서비스가 취소될 수 있습니다.

  • 웹서비스 용도로만 사용이 가능합니다.
  • 웹서비스 이외의 용도로 사용 시 라이선스 위반 사유로 서비스 계약이 해지될 수 있습니다.

  • 웹메일 무료제공

    별도의 사이트 개설이나 메일 서버 구축 없이 고객님의 도메인으로 고유 메일 주소를 이용할 수 있습니다. 고객님의 고유 메일 주소는 대형 포털에서 발급해 주는 무료 메일 주소가 할 수 없는 기업의 홍보역할을 합니다.

  • 그룹웨어 무료제공

    웹메일, 전자결재, 전자게시판, 업무관리 등 통합적 업무관리를 할 수 있는 Smart Work 솔루션을 경험할 수 있습니다. 추가 비용 없이, 우리 회사만의 메신저(Office Talk)를 함께 사용할 수 있습니다.

  • 웹로그 제공

    방문자 분석에서부터 광고효과분석까지 가능한 웹로그분석 ASP 서비스를 무료로 제공해 드립니다. 카페24 웹로그를 통해 지속적인 고객분석을 통해 효율적으로 사이트를 운영하세요.
