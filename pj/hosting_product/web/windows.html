  • 윈도우 단독웹호스팅은 마이크로소프트사의 Windows를 운영체제로 하고 있으며, ASP의 새로운 버전으로서 기업 클래스의 웹 응용프로그램을 만들고자 하는 개발자들에게 필요한 서비스들을 제공하는 통일된 웹 개발 플랫폼입니다.
  • ASP에서 사용되던 문법들을 사용할 수 있으며 Class를 사용하여 객체지향 프로그램이 가능하므로 보다 나은 개발환경과 프로그램 모듈 생성이 가능하며 기존의 ASP에서는 다룰 수 없었던 더욱 강력한 기능이 지원됩니다.
  • ASP.NET은 컴파일된 .NET프레임워크 기반 환경으로 C, C++, C#, VB, J# 등을 포함한 .NET 프레임워크 호환 가능언어로 응용 프로그램을 만들 수 있고, 다양한 언어의 통합, 보안성, 확장성, 관리 효율성 등의 장점이 있습니다.
  • 하나의 서버를 여러 사용자가 공동으로 사용하는 웹호스팅과 달리 초고속의 전송 속도와 풍부한 자원을 고객님 단독으로 이용할 수 있는 서비스입니다.
  • 개인이나 기업에서 각종 인터넷 서비스 제공으로 전용회선 및 서버를 구축하고, 전문 엔지니어 채용 및 자체 전산실 운영으로 발생할 수 있는 부담을 카페24 단독웹호스팅이 해결해 드립니다.
  • 웹호스팅 서비스와 같이 관리자툴(네트워크 모니터링, 전송량 통계 등) 제공으로 초보자도 쉽게 이용할 수 있습니다.
  • APM, IIS/MSSQL설치에서부터 튜닝, 보안 업데이트까지 전문 엔지니어가 24시간 365일 시스템을 관리해드립니다.
  • 백업 HDD를 함께 제공하므로 고객님의 소중한 데이터를 보다 안전하게 관리해 드립니다.
  • 네트웍 모니터링, 전송량 통계, HDD 사용량 실시간 체크 등 다양한 서버 모니터링을 제공합니다.
  • 장애 발생 시 신속하고 정확한 복구 서비스를 제공합니다.
  • 디스크 무료 추가 제공 (백업 디스크로 사용됩니다.)
  • MS-SQL DB는 기본 공용 서버로 제공됩니다.
  • 일반 FTP 프로그램에 익숙하지 않은 고객님들을 위해 간편하고 사용하기 쉬운 웹FTP를 지원합니다.
  • FTP란 호스팅을 받는 서버에 접근하여 파일을 업로드하거나 다운로드 할 수 있는 툴(TOOL)입니다.
자세히보기
  • 리눅스 서버에는 웹방화벽(ModSecurity)을 윈도우 서버에는웹나이트(WebKnig-ht)를 설치해 드립니다.
    • 웹방화벽(ModSecurity)
      • Apache 웹 서버 기반의 오픈 소스 방화벽으로 PHP Injection및 SQL Injection, XSS 등의 다양한 웹 공격 및 악의적인 접속요청이 발견되면 차단, 로깅 등의 사전에 정의된 행위를 수행하므로 안전한 호스팅 서비스를 받으실 수 있습니다.
    • 웹나이트(WebKinght)
      • IIS 웹 서버에 설치되는 공개용 웹방화벽으로 ISAPI 필터 형태로 동작하며, IIS 서버 최앞단에 위치하여 패킷이
        웹서버로 접근하기 전에 필터 룰에 따라 검증을 하고 SQL Injection공격 등 특정 웹 요청을 사전에 차단함으로서
        웹서버의 보안을 한층 더 강화할 수 있습니다.
  • 고객님의 사이트를 안전하게 지켜드리는 웹방화벽으로 안전한 호스팅 서비스를 받으실 수 있습니다.

초과 트래픽 요금
  • 단독웹호스팅은 약정된 네트워크 용량을 초과해서 사용하실 경우, 초과된 용량에 대해 한달 단위로 후불결제를 하고 있습니다.
  • 초과 네트워크(트래픽) 요금을 계산하는 방식
    • 계산 기준
      • ⓐ 사용기간(과금대상기간)의 5분단위 트래픽 측정값 중 최대값
      • ⓑ 사용기간(과금대상기간)의 총 평균 트래픽의 3배
    • 계산 기준
      • ⓐ, ⓑ 중 최소 트래픽 값을 계산 합니다.
      • 초과금액 = (총사용트래픽 - 기본트래픽) * 17,000원(1Mbps당) (VAT 별도)
