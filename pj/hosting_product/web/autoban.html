제공 환경

  • 서버환경 선택 가능
    • PHP 4 / MySQL 4.X / EUC-KR
    • PHP 5.3 / Mysql 5.x / EUC-KR
    • PHP 5.3 / Mysql 5.x / UTF-8
    • PHP 5.5 / Mysql 5.x / EUC-KR
    • PHP 5.5 / Mysql 5.x / UTF-8
    • PHP 7.0 / MariaDB-10.0.x / UTF-8 / HTTP2.0 / Apache2 / InnoDB 지원
    • 워드프레스, 제로보드XE, 그누보드 등 프로그램 자동설치 도움말 및 MySQi 지원
  • 무료 도메인 제공
    • 호스팅에 연결되는 아이디를 이용한 무료 도메인을 제공합니다.
    • 호스팅 신청 아이디가 myname인 경우 myname.cafe24.com 도메인을 무료로 제공해 드립니다.
  • MariaDB 제공
    • 서버환경을 PHP 7.0 으로 선택 시 MariaDB가 제공됩니다.
    • MariaDB는 MySQL보다 빠른 성능, 높은 호환성의 관계형 데이터 베이스 입니다.
    • 오픈 소스의 관계형 데이터베이스 관리 시스템(RDBMS)으로 MySQL 대비 최고 70% 이상의 성능이 향상되었습니다.
    • 새로운 저장 엔진인 아리아(Aria)뿐만 아니라 InnoDB를 교체할 수 있는 XtraDB 저장 엔진을 포함합니다.
    • MySQL과 동일한 소스 코드를 기반으로 하여, MySQL이 가지고 있는 모든 기능을 완벽히 구현하여 MySQL과 높은 호환성을 유지합니다.
  • FTP/SFTP/SSH 지원 도움말
    • SSH 터널링(Tunneling)은 지원하지 않습니다.
    • 보안강화를 위해 [나의서비스관리 > 보안관리 > FTP/Shell 접속설정]에서 필요에 따라 on/off 및 설정이 가능합니다.
  • SSL 향상
    • SNI 지원합니다.
    • SSL 기본포트 (443) 사용가능합니다.
  • Language/Script
    • cgi, perl script, gcc, C/C++ Complier, SSI, Flash, DBI/DBD, zend optimizer, Python, GD라이브러리 등
    • 이외의 다른 lib나 파일을 설치해서 사용할 시에는 작동을 안하거나 비정상적으로 작동할 수 있습니다.

자세히보기
  • 각 계정의 메일 용량은 5M이며, 메일 발송은 한번에 30통/ 하루에 500통까지만 발송이 가능합니다.
  • 500통이 초과되면 발송은 차단되고, 자정(00시)에 초기화 됩니다.
  • 일반 FTP 프로그램에 익숙하지 않은 고객님들을 위해 간편하고 사용하기 쉬운 웹FTP를 지원합니다.
  • FTP란 호스팅을 받는 서버에 접근하여 파일을 업로드하거나 다운로드 할 수 있는 툴(TOOL)입니다.
자세히보기
  • 고객님들이 간편하고 쉽게 DB를 관리하실 수 있도록 MySQL 웹어드민을 제공합니다.
  • MySQL이란 리눅스 기반의 호스팅 서버에서 사용되는 DB(데이터베이스) 입니다.
자세히보기
  • 리눅스 서버에는 웹방화벽(ModSecurity)을 윈도우 서버에는웹나이트(WebKnig-ht)를 설치해 드립니다.
    • 웹방화벽(ModSecurity)
      • Apache 웹 서버 기반의 오픈 소스 방화벽으로 PHP Injection및 SQL Injection, XSS 등의 다양한 웹 공격 및 악의적인 접속요청이 발견되면 차단, 로깅 등의 사전에 정의된 행위를 수행하므로 안전한 호스팅 서비스를 받으실 수 있습니다.
    • 웹나이트(WebKinght)
      • IIS 웹 서버에 설치되는 공개용 웹방화벽으로 ISAPI 필터 형태로 동작하며, IIS 서버 최앞단에 위치하여 패킷이 웹서버로 접근하기 전에 필터 룰에 따라 검증을 하고 SQL Injection공격 등 특정 웹 요청을 사전에 차단함으로서 웹서버의 보안을 한층 더 강화할 수 있습니다.
  • 고객님의 사이트를 안전하게 지켜드리는 웹방화벽으로 안전한 호스팅 서비스를 받으실 수 있습니다.
자세히보기
  • 호스팅을 신청하시면 인공지능 스팸필터가 지긋지긋한 스팸 게시물에서 해방시켜 드립니다.
  • 제로보드, 테터툴즈 뿐만 아니라 그외 어느 게시판을 클릭 한번으로 스팸필터를 작동시킬 수 있어 편리합니다.

  • 웹메일 무료제공

    별도의 사이트 개설이나 메일 서버 구축 없이 고객님의 도메인으로 고유 메일 주소를 이용할 수 있습니다. 고객님의 고유 메일주소는 대형 포털에서 발급해주는 무료 메일 주소가 할 수 없는 기업의 홍보역할을 합니다.

  • 그룹웨어 무료제공

    웹메일, 전자결재, 전자게시판, 업무관리 등 통합적 업무관리를 할 수 있는 Smart Work 솔루션을 경험할 수 있습니다. 추가 비용 없이, 우리 회사만의 메신저(Office Talk)를 함께 사용할 수 있습니다.

  • 이미지 CDN 무료 제공 도움말

    카페24 호스팅을 신청하시면 무료로 이미지 CDN 서비스가 제공되어 사실적 용량 증가 효과가 있습니다. 트래픽이 많이 발생하는 이미지를 무료 이미지 CDN 서버에서 제공함으로써 웹 페이지에 집중되는 트래픽을 분산시킬 수 있습니다.

  • 스트리밍 무료 제공 도움말

    카페24 호스팅을 신청하시면 무료로 스트리밍 서비스가 제공되어 사실적 용량 증가 효과가 있습니다. 음악파일 또는 동영상으로 인한 트래픽 증가로 스트리밍 서비스가 차단되더라도 사이트의 차단없이 안정적으로 운영하실 수 있습니다.

  • 웹로그 제공

    방문자 분석에서부터 광고효과 분석까지 가능한 웹로그분석 ASP 서비스를 무료로 제공해 드립니다. 카페24 웹로그를 통해 지속적인 고객분석을 통해 효율적으로 사이트를 운영하세요.
