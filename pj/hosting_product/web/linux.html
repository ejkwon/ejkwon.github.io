제공 환경

  • 서버환경 (선택 가능)
    • 오픈 소스 기반의 리눅스를 운영체제로 하고 있으며 카페24 호스팅 서비스에서 안정성이 입증된 APM 환경을 제공합니다.
    • Apache2.4/PHP7+Mariadb-10.x UTF-8 : 제로보드XE,워드프레스 자동설치,다국어 사이트 적합
    • Apache1.3/PHP5.3x,MySQL5.1x UTF-8 : 제로보드XE,워드프레스 자동설치,다국어 사이트 적합
      아래 서버 구성 중 선택 가능합니다.
      • Apache+PHP+MySQL
      • Apache+PHP+MySQL(InnoDB)
      • Apache+Tomcat7+PHP+MySQL
    • Apache1.3, PHP5.3x, MySQL5.1x EUC-KR : 제로보드 4, 그누보드 설치 가능
    • Apache1.3, PHP4.4x, MySQL4.0x EUC-KR : 제로보드 4, 그누보드 설치 가능
    • 이미지호스팅 전용 : 이미지/ html 서비스에 최적화, Apache(웹서버)만 설치되어 제공
  • 무료 도메인 제공
    • 호스팅에 연결되는 아이디를 이용한 무료 도메인을 제공합니다.
    • 호스팅 신청 아이디가 myname인 경우 myname.cafe24.com 도메인을 무료로 제공해 드립니다.
  • SFTP/FTP/SSH 지원 도움말
    • SSH 터널링(Tunneling)은 지원하지 않습니다.
  • SSL 향상
    • SNI 지원합니다.
    • SSL 기본포트 (443) 사용가능합니다.
  • Language/Script
    • cgi, perl script, gcc, C/C++ complier, SSI, Flash, PHP4, DBI/DBD, zend optimizer, Python, GD 라이브러리 등
  • 기타
    • 백본(backbone)에서 고객님의 서버까지 1Gbps dedicated 라인으로 연결
    • 단독웹호스팅 서비스는 영구임대 상품이며, root 권한은 제공되지 않습니다.
      root 권한 제공 상품 바로가기
    • 웹서버와 DB서버 분리를 통해 보다 안정적인 사이트 운영이 가능합니다. (옵션 선택)
    • 단독웹호스팅은 서버 내 1계정이며, 추가 계정은 지원하지 않습니다.

  • FTP란 호스팅을 받는 서버에 접근하여 파일을 업로드하거나 다운로드 할 수 있는 툴(TOOL)입니다.
자세히보기
  • 고객님들이 간편하고 쉽게 DB를 관리하실 수 있도록 MySQL 웹어드민을 제공합니다.
  • MySQL이란 리눅스 기반의 호스팅 서버에서 사용되는 DB(데이터베이스) 입니다.
자세히보기
  • 리눅스 서버에는 웹방화벽(ModSecurity)을 윈도우 서버에는웹나이트(WebKnig-ht)를 설치해 드립니다.
    • 웹방화벽(ModSecurity)
      • Apache 웹 서버 기반의 오픈 소스 방화벽으로 PHP Injection및 SQL Injection, XSS 등의 다양한 웹 공격 및 악의적인 접속요청이 발견되면 차단, 로깅 등의 사전에 정의된 행위를 수행하므로 안전한 호스팅 서비스를 받으실 수 있습니다.
    • 웹나이트(WebKinght)
      • IIS 웹 서버에 설치되는 공개용 웹방화벽으로 ISAPI 필터 형태로 동작하며, IIS 서버 최앞단에 위치하여 패킷이
        웹서버로 접근하기 전에 필터 룰에 따라 검증을 하고 SQL Injection공격 등 특정 웹 요청을 사전에 차단함으로서
        웹서버의 보안을 한층 더 강화할 수 있습니다.
  • 고객님의 사이트를 안전하게 지켜드리는 웹방화벽으로 안전한 호스팅 서비스를 받으실 수 있습니다.
자세히보기
  • 호스팅을 신청하시면 인공지능 스팸필터가 지긋지긋한 스팸 게시물에서 해방시켜 드립니다.
  • 제로보드, 테터툴즈 뿐만 아니라 그외 어느 게시판을 클릭 한번으로 스팸필터를 작동시킬 수 있어 편리합니다.

초과 트래픽 요금
  • 단독웹호스팅은 약정된 네트워크 용량을 초과해서 사용하실 경우, 초과된 용량에 대해 한달 단위로 후불결제를 하고 있습니다.
  • 초과 네트워크(트래픽) 요금을 계산하는 방식
    • 계산 기준
      • ⓐ 사용기간(과금대상기간)의 5분단위 트래픽 측정값 중 최대값
      • ⓑ 사용기간(과금대상기간)의 총 평균 트래픽의 3배
    • 계산 방식
      • ⓐ, ⓑ 중 최소 트래픽 값을 계산 합니다.
      • 초과금액 = (총사용트래픽 - 기본트래픽) * 17,000원(1Mbps당) (VAT 별도)

  • 웹메일 무료제공

    별도의 사이트 개설이나 메일 서버 구축 없이 고객님의 도메인으로 고유 메일 주소를 이용할 수 있습니다. 고객님의 고유 메일 주소는 대형 포털에서 발급해 주는 무료 메일 주소가 할 수 없는 기업의 홍보역할을 합니다.

  • 그룹웨어 무료제공

    웹메일, 전자결재, 전자게시판, 업무관리 등 통합적 업무관리를 할 수 있는 Smart Work 솔루션을 경험할 수 있습니다. 추가 비용 없이, 우리 회사만의 메신저(Office Talk)를 함께 사용할 수 있습니다.

  • 이미지 CDN 무료 제공 도움말

    카페24 호스팅을 신청하시면 무료로 이미지 CDN 서비스가 제공되어 사실적 용량 증가 효과가 있습니다. 트래픽이 많이 발생하는 이미지를 무료 이미지 CDN 서버에서 제공함으로써 웹 페이지에 집중되는 트래픽을 분산시킬 수 있습니다.

  • 스트리밍 무료 제공 도움말

    카페24 호스팅을 신청하시면 무료로 스트리밍 서비스가 제공되어 사실적 용량 증가 효과가 있습니다. 음악파일 또는 동영상으로 인한 트래픽 증가로 스트리밍 서비스가 차단되더라도 사이트의 차단없이 안정적으로 운영하실 수 있습니다.

    방문자 분석에서부터 광고효과분석까지 가능한 웹로그분석 ASP 서비스를 무료로 제공해 드립니다. 카페24 웹로그를 통해 지속적인 고객분석을 통해 효율적으로 사이트를 운영하세요.
