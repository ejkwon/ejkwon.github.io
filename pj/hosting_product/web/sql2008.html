제공 환경

  • 서버환경
    • 운영체계 : Windows Server 2008 R2 64bit
    • DB : MS-SQL 2008 R2 64bit
    • NET Framework : 4.0
  • 무료 도메인 제공
    • 호스팅에 연결되는 아이디를 이용한 무료 도메인을 제공합니다.
    • 호스팅 신청 아이디가 myname인 경우 myname.cafe24.com 도메인을 무료로 제공해 드립니다.
  • Language/Script
    • ASP, .NET, C#, XML, HTML, Flash, Javascript, Vbscript (PHP, CGI 사용불가)
  • FTP 지원 도움말
    • 계정 아이디별로 포트를 별도 할당합니다. Telnet은 지원하지 않습니다.
  • MS-SQL 접속
    • SQL Server Management Studio로 접속하셔야 합니다.
  • 웹(http)으로 접근
    • 반드시 www 폴더 내에 업로드 또는 새폴더를 생성해 주셔야 합니다.
  • 컴포넌트
    • Upload 컴포넌트 : DEXTUpload Pro x64
    • 실명인증 컴포넌트 : 한국신용평가
  • 결제모듈 지원
    • 이니시스, KCP

제공 기능

  • 고성능 지원
  • 보안 지원
  • 개발 지원
  • 관리 지원
  • 이메일
    계정
  • 웹FTP 제공
  • 웹방화벽
    제공
  • 대용량 데이터 처리에 대한 신속한 응답 제공
  • 중요 데이터를 보호하는 강력한 보안 제공
  • 비즈니스 확장이 용이한 데이터 인프라 구성 지원
  • 쿼리 실행과 인덱스 작업 병렬처리 제공
  • NUMA(Non-Uniform Memory Access) 지원으로 시스템 성능 향상 가능
  • 자원 관리자(Resource Governor)를 이용하여 서버자원을 효과적으로 분배 지원
  • 정책 기반의 보안 관리 제공
  • 이벤트 및 쿼리에 대한 중앙집중적 통합 감사 제공
  • 데이터 및 로그파일, 백업 파일에 대한 암호화 지원
  • 다양한 데이터를 효율적으로 저장하는 데이터 통합 저장소 제공
  • 빠르고 쉬운 애플리케이션 개발 가능
  • T-SQL, C#, VB.NET 등 다양한 언어로 개발 가능
  • 공간 데이터를 지원하여 응용 프로그램을 통해 장소 기반 데이터 사용 가능
  • Performance Studio를 사용하여 데이터 서비스 모니터링 통합 관리 제공
  • 각 계정의 메일 용량은 20M이며, 메일 발송은 한번에 30통/ 하루에 500통까지만 발송이 가능합니다.
  • 500통이 초과되면 발송은 차단되고, 자정(00시)에 초기화 됩니다.
  • 일반 FTP 프로그램에 익숙하지 않은 고객님들을 위해 간편하고 사용하기 쉬운 웹FTP를 지원합니다.
  • FTP란 호스팅을 받는 서버에 접근하여 파일을 업로드하거나 다운로드 할 수 있는 툴(TOOL)입니다.
자세히보기
  • 웹나이트(WebKinght)를 기본으로 제공합니다.
  • IIS 웹 서버에 설치되는 공개용 웹방화벽으로 ISAPI 필터 형태로 동작하며, IIS 서버 최앞단에 위치하여 패킷이 웹서버로 접근하기 전에 필터 룰에 따라 검증을 하고 SQL Injection공격 등 특정 웹 요청을 사전에 차단함으로서 웹서버의 보안을 한층 더 강화할 수 있습니다.

서비스 이용
월사용료 혜택
  • 신청하시는 서비스 기간에 따라 월사용료 할인 혜택을 제공합니다.

    (하위 서비스로의 변경은 불가능)
서비스 제한
  • 서버 부하가 과도한 계정의 경우 서비스가 취소될 수 있습니다.
  • 64bit 모듈만 지원되며, 32bit는 지원하지 않습니다.

  • 웹메일 무료제공

    별도의 사이트 개설이나 메일 서버 구축 없이 고객님의 도메인으로 고유 메일 주소를 이용할 수 있습니다. 고객님의 고유 메일 주소는 대형 포털에서 발급해 주는 무료 메일 주소가 할 수 없는 기업의 홍보역할을 합니다.

  • 그룹웨어 무료제공

    웹메일, 전자결재, 전자게시판, 업무관리 등 통합적 업무관리를 할 수 있는 Smart Work 솔루션을 경험할 수 있습니다. 추가 비용 없이, 우리 회사만의 메신저(Office Talk)를 함께 사용할 수 있습니다.

  • 웹로그 제공

    방문자 분석에서부터 광고효과분석까지 가능한 웹로그분석 ASP 서비스를 무료로 제공해 드립니다. 카페24 웹로그를 통해 지속적인 고객분석을 통해 효율적으로 사이트를 운영하세요.
