mButton

  • mButton 기본적으로 가운데 정렬이고, mButton+grid1(2~4) 로 Full 사이즈 영역의 크기별 표현이 가능하다. (사파리%기준)
  • 페이지 하단의 mButton 내에는 일반적으로 btnSave / btnClose 버튼을 적용한다.
