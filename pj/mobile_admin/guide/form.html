Form / Grid / Font / Img

Form > input

  • fChk : checkbox / radio
  • fText : text / password (넓이는 각각 인라인 스타일로 적용)
  • fText는 기본 왼쪽정렬로, 중앙정렬 필요시 center, 오른쪽정렬 필요시 right 클래스를 추가한다.

Form > textarea

  • fTextarea : 텍스트 영역
  • rows, cols 속성을 필수로 적용하고, 기본 width:100%가 아닌 경우, 인라인 스타일로 재 설정한다.

Form > select

  • fSelect : 기본 셀렉트
  • fSelect half : 한 줄의 절반을 차지(사파리%기준)
  • fSelect full : 넓이 100%

Form > mSelect

  • 일반 선택폼인 fSelect와는 다르게 더 직관적으로 선택하는 영역으로, 커스텀 셀렉트박스 mSelect 를 사용한다.
  • 가로 100%를 기본으로, mSelect+grid2(3~4) 추가하여 Full 사이즈 영역의 갯수만큼 사용가능하다. (사파리%기준)

Grid

  • gSingle / gDouble / gTriple / gIndent / gSpace
  • 컨텐츠 사이의 여백만 처리할 경우, span 이나 p 태그를 사용해서 적용한다.

Font > txt

  • txtNormal / txtEm / txtWarn / txtBasic / txtStrong / txtLight / txtLess
  • td나 li등에 직접 class처리하지 않고, 반드시 inline element 로 감싸준다. (span, strong 등)

Font > txtInfo

  • 폼태그의 안내문구로, p태그 ul 태그 사용 가능

Font > help

  • 기본 도움말로, p태그 ul 태그 사용 가능
  • help typeBorder : 테두리선이나 여백 필요시 추가한다.

img

  • 테두리선 필요시 frame 클래스를 적용한다.

  • [ul li태그] 폼태그의 2줄 이상 안내문구입니다.
