mSummary

  • mSummary 내에 셀 구분이 필요한 경우, cell / info 클래스로 구분한다.
  • cell 클래스에 해당 키워드로 다중클래스를 추가하여, width 크기요소를 필수로 적용한다. (예시) .mSummary .cell.thumb { width:50px; }
  • 기본 디자인과 다를 경우, mSummary 클래스에 g$Class 다중 클래스를 추가하여 사용한다. (예시) .mSummary.gMove .cell.thumb img { margin:0; }
  • mSummary 내에 a태그 범위는 목록 전체를 기본으로 하되, 부분 영역 표시가 필요한 경우에는 예외로 처리한다.
  • mSummary 내에 heading 클래스는 주요 제목에 부여하여 문구가 길면 말줄임 처리하고,
    description 클래스는 서술적인 내용으로써 줄바꿈 처리한다.
