타이틀

mTitle

타이틀은 mTitle 내에 타이틀 h1 / 뒤로가기 버튼 / 전송 버튼으로 구성한다.

  • 뒤로가기 버튼 : btnTitle+back
  • 전송 버튼 : btnTitle+submit
  • 서브 레이아웃의 타이틀은 헤더영역 고정이므로 "대제목"으로 통일한다.
  • 메인 레이아웃에서 타이틀이 필요한 경우는 mTitle 부분을 contents 내 상단에 적용한다. [예시]
