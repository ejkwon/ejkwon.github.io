신청 즉시 사용

cafe24만의 기술력으로 신청 즉시 서버 접속 및 이용이 가능합니다.

고객님이 별도의 APM 설치 및 세팅이 필요없습니다.

서버 모니터링 툴 제공

cafe24의 서비스를 이용하는 고객님을 위하여 트래픽 모니터링, 서버 성능 그래프, 웹 성능 그래프, DB 성능 그래프 등 운영 중인 내 서버의 현황을 실시간으로 모니터링할 수 있는 서비스를 무료로 제공합니다.

서버 원격 제어

원격 또는 로컬 서버의 상태를 확인하고 이를 직접 제어할 수 있는 기능을 제공합니다.

IPMI 기능을 이용하면 OS설치 등을 위해 IDC에 방문할 필요 없이 원격지에서 직접 서버를 관리하고 사용할 수 있습니다.

하이퍼 스레딩

인텔® 하이퍼 스레딩 기술(인텔® HT 기술)은 물리적 코어당 2개의 논리적인 처리 스레드를 제공합니다.

싱글 스레드보다 처리 속도가 빠르며, 멀티스레드를 지원 하는 소트프웨어에서 더욱 향상된 시스템 성능을 제공합니다.

Full SSD 장착

업계 최초로 DATA와 DB 영역까지 고성능 SSD 디스크를 제공 합니다.

SSD 의 Full 장착으로, 고용량 데이터 처리에 탁월합니다.

포트모니터링 기능 제공

설정된 포트가 응답이 없을 때 메일 및 SMS로 알려드립니다.

웹 모니터링

홈페이지가 다운되어 http 응답이 비정상적일 때 SMS로 알려드립니다.

무료 그룹웨어/웹메일 서비스

그룹웨어 3GB + 웹하드 1GB + 웹메일500MB 의 용량을 제공합니다.
