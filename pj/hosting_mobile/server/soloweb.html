단독 이용 웹호스팅

하나의 서버를 여러 사용자가 공동으로 사용하는 웹호스팅과 달리 초고속의 전송 속도와 풍부한 자원을 고객님 단독으로 이용할 수 있는 서비스입니다.

안정적인 호스팅 환경

국내 최대 백본망을 보유한 KT-IDC와 첨단 네트워크를 자랑하는 SK-IDC에 입주하여 안정되고 빠른 최상의 서비스를 제공합니다.

24시간 365일 시스템 관리

APM, IIS/MSSQL설치에서부터 튜닝, 보안 업데이트까지 전문 엔지니어가 24시간 365일 시스템을 관리해드립니다.

리눅스-웹방화벽(ModSecurity)
기본 제공

Apache 웹 서버 기반의 오픈 소스 방화벽으로 PHP Injection및 SQL Injection, XSS 등의 다양한 웹 공격 및 악의적인 접속요청이 발견되면 차단, 로깅 등의 사전에 정의된 행위를 수행하므로 안전한 호스팅 서비스를 받으실 수 있습니다.

윈도우-웹나이트(WebKinght)
기본 제공

IIS 웹 서버에 설치되는 공개용 웹방화벽으로 ISAPI 필터 형태로 동작하며, IIS 서버 최앞단에 위치하여 패킷이 웹서버로 접근하기 전에 필터 룰에 따라 검증을 하고 SQL Injection공격 등 특정 웹 요청을 사전에 차단함으로서 웹서버의 보안을 한층 더 강화할 수 있습니다.

스트리밍 무료제공

음악파일 또는 동영상으로 인한 트래픽 증가로 스트리밍서비스가 차단되더라도 홈페이지의 차단없이 안정적인으로 운영하실 수 있습니다.

이미지 CDN 무료제공

트래픽이 많이 발생하는 이미지를 무료 이미지 CDN 서버에서 제공함으로써 웹 페이지에 집중되는 트래픽을 분산시킬 수 있습니다.

게시판 스팸필터 제공

제로보드, 테터툴즈 뿐만 아니라 그외 어느 게시판을 클릭 한번으로 스팸필터를 작동시킬 수 있어 편리합니다.
