국내 최고 시설과 안정성을 자랑하는 목동 KT IDC에서 고객님의 서버를 관리합니다.

서버 통합관리 서비스

OS 설치에서부터 튜닝,버전 업그레이드 등의 사후관리까지 카페24 시스템 엔지니어가 1:1로 관리해 드립니다.

로드밸런싱으로
끊김없는 서비스

LVS,L4 스위치를 이용한 로드밸런싱은 24시간, 365일 서버부하 및 장애 걱정 없이 안정적인 서버 운영을 가능하게 합니다.

24시간 보안 모니터링 서비스

카페24는 24시간 깨어있는 보안모니터링 서비스를 제공합니다.

보안관제,취약점 점검,모의해킹/침해사고탐지 등 상시 모니터링을 통해 고객님의 서버를 안전하게 지켜드립니다.

서버 모니터링 툴 제공

cafe24의 서비스를 이용하는 고객님을 위하여 트래픽 모니터링, 서버 성능 그래프, 웹 성능 그래프, DB 성능 그래프 등 운영 중인 내 서버의 현황을 실시간으로 모니터링할 수 있는 서비스를 무료로 제공합니다.

포트모니터링 기능 제공

설정된 포트가 응답이 없을 때 메일 및 SMS로 알려드립니다.

웹 모니터링

홈페이지가 다운되어 http 응답이 비정상적일 때 SMS로 알려드립니다.

무료 그룹웨어/웹메일 서비스

그룹웨어 3GB + 웹하드 1GB + 웹메일500MB 의 용량을 제공합니다.
