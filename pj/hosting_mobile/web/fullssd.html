10Gbps 광(光)네트워크
빠른 네트워크 서비스를 제공하기 위해 10Gbps 네트워크 인프라 구축

네트워크는 대역폭이 넓어야 합니다.

순간적인 대규모 트래픽이 유입되더라도 안전한 네트워크 서비스를 제공합니다.

광(光)랜카드를 이용해 각 서버마다 10Gbps 광(光) Dedicated Line 으로 연결하여 빠르고 안정적인 서비스를 제공합니다.

차세대 저장장치 Full SSD장착
DATA와 DB 영역까지 고성능 SSD가 장착된 Full SSD 서버
혁신적인 데이터 처리 속도
서버의 성능을 획기적으로 높이는 SSD 의 Full 장착으로, 고용량 데이터 처리에 탁월합니다.
뛰어난 내구성으로 안정성 획득
소비 전력이 현저히 낮고 충격에도 강하여, 안정성이 더욱 보장됩니다.
총 8개의 엔진! 고성능 CPU
Quad-Core CPU(*2)=8Core, 쿼드코어64bit 고사양서버

리눅스 기반 웹호스팅으로 폭발적인 사양의 고성능 서버와 차세대 기술로 더욱 빠르고 안정적인 호스팅을 제공합니다.

쿼드쿼어 Quad-Core 란?
1개의 CPU에 4개의 코어가 탑재된 것으로, 아우토반 호스팅 서버 1대에 CPU가 2개로 구성되어, 총 8개의 CPU 처리 능력을 가지고 있는 폭발적인 사양입니다.
MySQL 기본제공+추가 DB지원

사용자의 환경에 맞게 다양한 DB를 지원합니다.

기본적으로 리눅스 환경의 MySQL을 제공하고 추가적으로 사용자가 원할 경우에는 Cubrid, PGSQL DB를추가 이용 가능합니다.

웹메일 무료 제공

별도의 사이트 개설이나 메일 서버 구축 없이 고객님의 도메인으로 고유 메일 주소를 이용할 수 있습니다.

고객님의 고유 메일 주소는 대형 포털에서 발급해 주는 무료 메일 주소가 할 수 없는 기업의 홍보역할을 합니다.
