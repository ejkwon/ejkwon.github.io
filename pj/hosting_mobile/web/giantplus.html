10Gbps 광(光)네트워크
빠른 네트워크 서비스를 제공하기 위해 10Gbps 네트워크 인프라 구축

네트워크는 대역폭이 넓어야 합니다.
순간적인 대규모 트래픽이 유입되더라도 안전한 네트워크 서비스를 제공합니다.

광(光)랜카드를 이용해 각 서버마다 10Gbps 광(光) Dedicated Line 으로 연결하여 빠르고 안정적인 서비스를 제공합니다.

64core (16core x4) CPU
16Core CPU(*4)=64Core,고사양서버

1개의 CPU에 16개의 코어가 탑재된 것으로 서버 1대에 CPU가 4개로 구성. 총 64개의 CPU 처리를 가진 고성능 서버입니다.

코어 수가 많아지면 데이터를 큰 단위로 처리 분석하는데 효율이 높고 신속한 실행으로 부하를 관리합니다.
