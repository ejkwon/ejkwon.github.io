Tomcat JSP 광호스팅소개

자바 기반의 웹사이트 개발기술인 JSP 이용을 위한 웹호스팅입니다.

JSP(Java Server Pages)는 Java 기반의 기술로 특정 플랫폼에 제한되지 않고 Java가 가지고 있는 효율성, 메모리 관리 등의 장점을 그대로 가지고 있는 강력한 언어입니다.

Tomcat 7.0 최신 버전 지원 및 단독 톰캣 서버 제공으로 보다 안정된 JSP호스팅을 제공합니다. 절약형부터 자이언트형까지 모든 상품에 단독톰캣서버를 제공합니다.

  • DB : MySQL
  • 개발언어 : JSP
  • Tomcat version : 8.0x/7.0.x
  • Servlet/JSP : 3.1/2.3, 3.0/2.2
  • JDK
단독 톰캣 서버의 장점
  • 고객님이 원하는 시간에 직접 시작/중지가 가능하여 수정된 사항이 사이트에 즉시 반영됩니다.
  • 톰캣 프로세스를 단독으로 사용할 수 있어 서비스가 더욱 안정적입니다.
  • 서블릿 재구동을 직접할 수 있습니다.
  • 공용서버의 주기적인 톰캣 재시작으로 인한 불편함이 없습니다.
  • 수정된 클래스 파일을 적용하기 위해 톰캣이 재시작되는 특정 시간을 기다려야 하는 공용 서버의 불편함이 없습니다.
  • 관리 편의를 위해 Tomcat Web Application Manager와 Virtual Host Manager를 지원합니다.
웹메일 무료 제공

별도의 사이트 개설이나 메일 서버 구축 없이 고객님의 도메인으로 고유 메일 주소를 이용할 수 있습니다.

고객님의 고유 메일 주소는 대형 포털에서 발급해 주는 무료 메일 주소가 할 수 없는 기업의 홍보역할을 합니다.
