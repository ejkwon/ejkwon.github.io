도메인 직접입력 연장/복구

  • 도메인을 직접 입력하시면 로그인 없이 도메인 연장이 가능합니다.

도메인 연장 시 결제 안내

  • 도메인은 연장완료 후에는 취소가 불가능합니다.
  • 세금계산서/현금영수증 안내
    신용카드 결제
    신용카드 매출전표가 세금계산서를 대체합니다.
    무통장 입금
    무통장 결제 시 선신청된 영수증(세금계산서 또는 현금영수증)은, 입금완료 후 익일 발행됩니다.
  • 단, 로그인을 하지 않은 결제이므로 입금완료 후 영수증 후신청은 불가능하오니 영수증 발행을 원하시면 결제 시 선신청하여 주세요.

도메인 복구 안내

  • 만료일로부터 30일째 되는 도메인은 삭제대기 상태가 되어 복구를 통해서 연장이 가능합니다.
  • 복구 가능 도메인 : .com | .net | .org | .biz | .info | .cc | .tv
  • 복구 비용 : 도메인 기간별 연장 비용(도메인마다 다름) + 복구비용(77,000원)

미등록된 도메인을 연장하실 경우

  • 검색하신 도메인이 미등록 상태인 경우에는 신규 등록을 통해 도메인을 등록해 주세요.
