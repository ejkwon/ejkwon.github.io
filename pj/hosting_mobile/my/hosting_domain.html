도메인 연결
  • 타사에서 도메인을 구입하신 경우, 도메인 구입처에서 네임서버를 변경해 주셔야 합니다.

도메인 연결 적용 안내

1. 도메인의 네임서버가 카페24 호스팅으로 설정되어 있을 경우

  • 연결 후 30분이 지나면 적용됩니다. 카페24 네임서버 보기

2. 보유 도메인을 연결할 경우

  • 도메인의 네임서버를 카페24로 변경해 주셔야 합니다.
  • 네임서버가 변경이 완료되는 데는 약 24시간~48시간이 소요됩니다.

3. 가상서버 호스팅의 경우

  • 도메인 추가연결은 카페24 호스팅 네임서버를 사용하시는 경우에만 사용이 가능합니다. 자체 도메인으로 네임서버를 운영하시는 경우에는 도메인 연결을 하시더라도 고객님 서버IP와 매핑되지 않습니다.
  • 도메인의 네임서버가 카페24 호스팅 네임서버로 설정되어 있는 경우 연결 후 15~20분이 지나면 도메인은 고객님의 서버IP와 매핑됩니다.
  • 연결된 도메인의 A 레코드는 고객님의 가상서버호스팅 IP가 되며, 계정과의 연결은 별도 설정을 하셔야 합니다.

4. 주의사항

  • 본인 소유의 도메인을 연결하실 수 있습니다. 타인 소유일 경우 소유자의 허락을 얻은 도메인을 연결하여 주십시오.
  • 타인 소유의 도메인으로 연결 시 분쟁이 발생할 경우 불이익을 당할 수 있습니다.

도메인 연결 삭제시 보안서버 SSL 인증서 주의할 점

1. 절약형/이코노미

  • 보안서버 SSL인증서를 신청한 도메인을 삭제하시면 인증서를 사용하실 수 없으며, 재연결 시 신규로 인증서를 구매하셔야 합니다.

2. 일반형/스탠다드/프로/프리미엄

  • 보안서버 SSL인증서가 설치된 도메인을 연결 도메인에서 '연결삭제'할 경우 인증서가 삭제됩니다.
  • 인증서를 설치하시려면 인증서가 발급된 도메인을 보유도메인으로 연결한 다음 [인증서관리] 메뉴의 [인증서받기]에서 인증서를 설치하시면 됩니다.
