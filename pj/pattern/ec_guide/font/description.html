관련 설명

  1. 1. 해당 페이지로 전체 브라우저에서의 폰트 확인이 가능합니다.
  2. 2. 폰트 우선순위에 따라 테이블 열 순서대로 노출됩니다.
  3. 3. 폰트 권장 사이즈는 아래 테이블의 배경색으로 표시된 행을 참고해주세요.
  4. 4. 폰트 확인 대상
    • PC
    • 모바일
    • 반응형
      • 모바일 사이즈로 PC에서 볼때
      • 모바일 사이즈를 모바일에서 볼때
  5. 5. 해외언어 lang 필수 적용
    • HTML 가이드
      • 한국어 : <html lang="ko">
      • 영어 : <html lang="en">
      • 일본어 : <html lang="ja">
      • 중국어 : <html lang="zh">
    • CSS 가이드: 일본어와 중국어는 폰트 최소사이즈 12px 처리
      • html:lang(ja) .txt10, html:lang(ja) .txt11 { font-size:12px; }
      • html:lang(zh) .txt10, html:lang(zh) .txt11 { font-size:12px; }
  6. 6. 폰트 사이즈 10~11px 필요할 경우, 전체 언어 공통으로 적용하기 위해 해당 클래스로 처리합니다.
    • <span class="txt10"> </span>
    • <span class="txt11"> </span>
